CSS 超出n(n=3)行隐藏显示省略...


      overflow : hidden;
      text-overflow: ellipsis;
      display: -webkit-box;
      -webkit-line-clamp: 3;
      -webkit-box-orient: vertical;

解决文本中汉字会自动换行，数字和字母不换行问题，添加属性： 


      word-break:break-all; // 会截断单词
      word-wrap:break-word; // 保留完整单词进行换行
